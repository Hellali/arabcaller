⚙️ بيانات الاتصال بـ Supabase

📋 الخطوة 1 — إنشاء مشروع Supabase

1. اذهب إلى supabase.com وسجّل مجاناً
2. اضغط New Project
3. اختر اسماً للمشروع (مثلاً: arabcaller)
4. اضبط كلمة مرور لقاعدة البيانات واحتفظ بها
5. انتظر دقيقتين حتى ينتهي الإعداد

📋 الخطوة 2 — إنشاء جداول قاعدة البيانات

من لوحة Supabase اضغط SQL Editor ثم الصق هذا الكود وشغّله:

-- جدول جهات الاتصال CREATE TABLE contacts ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, name text NOT NULL, phone text NOT NULL, specialty text, country text DEFAULT 'الجزائر', notes text, added_by text, added_at text, created_at timestamptz DEFAULT now() ); -- جدول المديرين CREATE TABLE admins ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, username text UNIQUE NOT NULL, password_hash text NOT NULL, full_name text NOT NULL, role text DEFAULT 'admin', created_at timestamptz DEFAULT now() ); -- السماح بالقراءة والكتابة العامة (الحماية عبر كلمة المرور في التطبيق) ALTER TABLE contacts ENABLE ROW LEVEL SECURITY; ALTER TABLE admins ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all_contacts" ON contacts FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all_admins" ON admins FOR ALL USING (true) WITH CHECK (true); -- إضافة المدير الرئيسي (كلمة المرور: admin123) INSERT INTO admins (username, password_hash, full_name, role) VALUES ( 'admin', '240be518fabd2724ddb6f04eeb1da5967448d7e831c08c8fa822809f74c720a9', 'المدير الرئيسي', 'superadmin' );

📋 الخطوة 3 — نسخ بيانات الاتصال

من لوحة Supabase اضغط Settings → API
ستجد:
• Project URL — انسخه في الحقل الأول أعلاه
• anon public key — انسخه في الحقل الثاني
